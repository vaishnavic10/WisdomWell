# BookSphere  

A console-based Library Management System built using Core Java, OOP, File Handling, DAO-Service Architecture.

## Features
- Add books and users
- Issue and return books
- View all books and users
- File-based persistent storage
- Custom exceptions
- Input validation
- Modular DAO + Service Layer

## Tech Stack
- Core Java
- OOP (Abstraction, Inheritance, Encapsulation, Polymorphism)
- Collections Framework
- File Handling
- JUnit (Testing)

## Project Structure
src/main/java/com/library/
  model/
  dao/
  service/
  exceptions/
  Main.java

src/main/resources/
  books.txt
  users.txt
  config.properties

## How to Run
javac Main.java  
java Main
